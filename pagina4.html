Desvio médio: é a média das distâncias absolutas de cada valor em relação à média do conjunto. Exemplo (simplificado): dados = 2, 4, 6 → média = 4 Distâncias: |2−4| = 2, |4−4| = 0, |6−4| = 2 → (2 + 0 + 2) ÷ 3 = 1,33

Variância: é a média dos quadrados dos desvios em relação à média. Mostra o quanto os dados se afastam da média. Exemplo: dados = 2, 4, 6 → média = 4 Desvios: (2−4)² = 4, (4−4)² = 0, (6−4)² = 4 → (4 + 0 + 4) ÷ 3 = 2,67

Desvio padrão: é a raiz quadrada da variância. Mede o “desvio típico” dos valores em relação à média, com a mesma unidade dos dados. Exemplo: √2,67 ≈ 1,63

Coeficiente de variação (CV): é o desvio padrão dividido pela média, geralmente expresso em porcentagem. Serve para comparar a variação entre conjuntos com médias diferentes. Fórmula: CV = ( Desvio Padrão / Média ) × 100 % Exemplo: desvio padrão = 1,63, média = 4 → (1,63 ÷ 4) × 100% ≈ 40,75%
