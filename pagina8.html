Determinante é um valor calculado a partir dos elementos de uma matriz quadrada, usado para resolver sistemas lineares e analisar propriedades da matriz.

Tipos de cálculo de determinantes:

  • Determinante de matriz 1×1: é o próprio elemento.
  • Determinante de matriz 2×2: Calcula-se como ad − bc, para matriz
    ab
    cd
  • Determinante de matriz 3×3: Pode ser calculado pela Regra de Sarrus ou pela expansão de Laplace.

    Regra de Sarrus para a matriz
    abc
    def
    ghi
    :
    det = aei + bfg + cdh − ceg − bdi − afh
  • Determinante de matrizes maiores: calculado pela expansão de Laplace, que consiste em escolher uma linha ou coluna e calcular a soma dos produtos dos elementos pelos seus respectivos cofatores.

    É uma técnica recursiva que utiliza determinantes de matrizes menores.
